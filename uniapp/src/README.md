![image](./static/images/live-chatroom.gif)


## 运行步骤

### 获取GoEasy Appkey
```
先注册一个账号，登录后，创建一个应用，就能得到您的appkey。
```
### 配置您的appkey
```
在main.js里，将appkey替换为您自己的common key
```


### 切换到NVUE
默认为vue, 可以在pages.json里，切换vue和nvue
